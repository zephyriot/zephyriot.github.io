.. title:: RAM_CONSOLE

.. option:: CONFIG_RAM_CONSOLE:
.. _CONFIG_RAM_CONSOLE:

Emit console messages to a RAM buffer "ram_console" which can
be examined at runtime with a debugger. Useful in board bring-up
if there aren't any working serial drivers.
