.. option:: CONFIG_PCI_LEGACY_BRIDGE:
.. _CONFIG_PCI_LEGACY_BRIDGE:

This option adds support for PCI legacy bridge device, that
allows direct setup of the PCI interrupt pin to IRQ number
mapping.
