.. title:: ETH_MCUX

.. option:: CONFIG_ETH_MCUX:
.. _CONFIG_ETH_MCUX:

Enable MCUX Ethernet driver.  Note, this driver performs one shot PHY
setup.  There is no support for PHY disconnect, reconnect or
configuration change.
