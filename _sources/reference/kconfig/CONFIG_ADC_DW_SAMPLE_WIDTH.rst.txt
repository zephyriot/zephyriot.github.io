.. title:: ADC_DW_SAMPLE_WIDTH

.. option:: CONFIG_ADC_DW_SAMPLE_WIDTH:
.. _CONFIG_ADC_DW_SAMPLE_WIDTH:

Defines ADC device data sample width (resolution):

- 0 = 6 bits resolution

- 1 = 8 bits resolution

- 2 = 10 bits resolution

- 3 = 12 bits resolution
