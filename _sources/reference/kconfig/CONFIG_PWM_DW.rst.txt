.. title:: PWM_DW

.. option:: CONFIG_PWM_DW:
.. _CONFIG_PWM_DW:

Enable driver to utilize PWM on the DesignWare Timer IP block.
Care must be taken if one is also to use the timer feature, as
they both use the same set of registers.
