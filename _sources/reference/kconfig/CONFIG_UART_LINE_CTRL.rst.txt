.. title:: UART_LINE_CTRL

.. option:: CONFIG_UART_LINE_CTRL:
.. _CONFIG_UART_LINE_CTRL:

This enables the API for apps to control the serial line,
such as baud rate, CTS and RTS.

Implementation is up to individual driver.

Says no if not sure.
