.. title:: NSIM

.. option:: CONFIG_NSIM:
.. _CONFIG_NSIM:

For running on nSIM simulator.

a) Uses non-XIP to run in RAM.
b) Linked at address 0x4000 with 0x4000 of RAM so that it works with
   a pc_size of 16 (default).
