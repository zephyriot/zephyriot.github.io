Arduino/Genuino 101 (Sensor Subsystem)

The Arduino 101 contains a 32 MHz Argonaut RISC Core (ARC)* EM processor as part of the Quark SE C1000 SoC within the Curie module. The ARC core is referenced as the digital signal processor (DSP) sensor hub or a sensor subsystem depending on what document youâ€™re looking at.

For more information about using the sensor subsystem with Zephyr, see Arduino/Genuino 101.
